# Case Studies Directive

## Purpose
Write structured case studies that demonstrate real AgentForge capabilities. Every case study must reference actual builds, not hypotheticals.

## Template

### Title: [Descriptive title — what was built and what it achieved]

**Problem**: What manual workflow was the target? Who had this pain? What did it cost them?

**Solution**: What did the AgentForge agent do? Which integrations? Which approval gates?

**Architecture**: Brief technical overview — agents involved, tools used, orchestration flow.

**Results**: Measurable outcomes — time saved, errors reduced, cost comparison, user feedback.

**Learnings**: What surprised us? What broke? What would we do differently?

## Publishing

- Long-form: Blog post or PDF
- Short-form: LinkedIn / X summary with link to full version
- Code: Link to relevant open-source components if applicable
